# restaurant-menu-order-system
## Description
A Java program that simulates a real-world self-order program with classes that represent the order (cart), the menu itself, etc.

## Classes
- Category.java: holds constant values of the different categories of items on the menu
- Size.java: holds constant values of the different sizes of the drinks
- Menu_Item.java: represents every item in the menu including its name, price, category and size
- Menu.java: represents the entire Menu itself
- Order.java: represents the real-world cart of the users items selected
- Restaurant.java: the main class where the program is executed

## How to Run
1. clone the repo
2. Compile all the .java files
3. Run the main (Restaurant.java)
